安装opencv-python库时卡住怎么解决（已解决）
----------

遇到 ModuleNotFoundError: No module named 'cv2' 解决办法

```
pip install opencv-python

```

如果运行上述命令进行opencv-python  
库安装时卡住，则继续下面的操作：

①更新[pip](https://edu.csdn.net/cloud/sd_summit?utm_source=glcblog&spm=1001.2101.3001.7020)版本：pip install --upgrade pip ，之后再次尝试

②当前的python版本不支持pip的版本（pip会直接装最新的opencv  
版本），指定安装某个版本就好啦\~例如我是3.6版本的python：

```
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python==4.3.0.38

```

```
     ③有些库下载时间较久，会误判成卡死，输入命令时，后面增加一个--verbose ，就可以将安装过程事无巨细的展现出来。例如：

```

```
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple --verbose

```

之后耐心等待即可。

方案②③是通用的，安装其他库的时候也适用。
